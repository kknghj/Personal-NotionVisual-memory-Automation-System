# Project Rules

이 프로젝트는 일반 아이콘 추천기가 아니라
사용자의 행동 회상 시스템이다.

사용자는:
- 카테고리
보다
- 실제 행동
- workflow
- 인터페이스 기억
- 서비스 UI 기억

을 우선한다.

중요 원칙:

- 메일 → 📧
- 전화 → 📞
- 카카오톡 → 💬
- 문서 작성/수정 → 📝
- 문서 검토/확인 → 📄
- 급여 시스템 입력 → 💰
- Excel 작업 → green spreadsheet icon
- Naver Form → green checkmark-circle
- 코딩 구현 → code
- 실행/환경설정 → terminal
- QR 인증 → qr-code

추천 시:
- 카테고리보다 행동을 우선
- 사람보다 인터페이스를 우선 가능
- workflow_priority를 고려
- 사용자 수정 기록을 GPT 판단보다 우선
